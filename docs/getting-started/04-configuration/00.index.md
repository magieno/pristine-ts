# Configuration

This chapter explains how Pristine apps declare and consume configuration values — database
URLs, feature flags, timeouts, anything that varies between environments. You'll learn the
declarative `configurationDefinitions` shape modules use, how the configuration manager
resolves values from your CLI invocation + environment variables + defaults, and how to
build custom resolvers for sources like AWS Parameter Store or HashiCorp Vault.
