# The Pristine CLI

The `pristine` binary is your day-to-day interface to a Pristine project. The Quick Start
chapter used it to scaffold and boot an empty app; this chapter explains every command,
the `pristine.config.ts` file that drives them, and how to write your own commands.

> **You'll learn**
> - What `pristine.config.ts` controls and how the CLI reads it
> - How `pristine build` writes a manifest so other commands know if the build is stale
> - How `pristine start` boots your app, handles signals, and shuts down gracefully
> - How `pristine verify` and `InstantiationTest` give you boot-time health checks
> - How to add your own commands
> - How `@commandParameter` declares flags, interactive prompts, choices, and error messages
> - How plugins let other packages ship commands into your CLI
