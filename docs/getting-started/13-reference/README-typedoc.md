# Pristine API Reference

This is the auto-generated symbol reference for every `@pristine-ts/*` package. It is
produced by [TypeDoc](https://typedoc.org/) from the TypeScript sources and accompanying
JSDoc comments under `packages/*/src/`.

## How to use this reference

- **Looking up a decorator or class by name?** Use the search box (top of the page).
- **Browsing one package?** Pick the package on the left-hand navigation.
- **Trying to learn how to *use* something?** This reference shows you the *shape*, not
  the *story*. For tutorials and examples, head back to the
  [Getting Started guide](../../index.md).

## Regenerating

```bash
npm run docs:reference
```

The script invokes TypeDoc through `npx` so no `npm install` is required just for docs.
The output is written into the same directory as this README
(`docs/getting-started/13-reference/api/`) and is git-ignored — regenerate locally or
in CI.

## Coverage

Every published `@pristine-ts/*` package is included. Excluded by default:
- `*.spec.ts` and `*.test.ts` files
- `private` members and `@internal` symbols
- Externally-defined types from `node_modules`
